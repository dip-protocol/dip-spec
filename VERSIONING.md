\# DIP Versioning



The Decision Integrity Protocol follows semantic versioning.



\## Version Format



MAJOR.MINOR.PATCH



Example:



1.0.0



\## Version Meaning



MAJOR  

Backward-incompatible protocol changes.



MINOR  

Backward-compatible feature additions.



PATCH  

Documentation or implementation fixes.



\## Artifact Version



Artifacts include an `artifact\_version` field that identifies the

protocol version used to construct the artifact.
