\# DIP Protocol Freeze — v1.0



Status: Frozen



This document declares the core Decision Integrity Protocol (DIP) v1 specification stable.



The following protocol components are considered immutable for v1:



\- Artifact schema

\- Decision schema

\- Proof schema

\- Canonicalization rules

\- Artifact ID derivation

\- Signing process

\- Verification procedure



Implementations MUST follow these rules to be considered DIP v1 compliant.



Future changes MUST be introduced through a new protocol version (v2).



\## Protocol Invariant



artifact + proof + verifier = truth



\## Version



DIP Protocol Version: 1.0
