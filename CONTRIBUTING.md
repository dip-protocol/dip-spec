\# Contributing to DIP



Thank you for contributing to the Decision Integrity Protocol.



All protocol changes must be proposed through the DIP proposal process.



\## Contribution Types



\- Protocol specification improvements

\- New DIP proposals

\- Documentation improvements

\- Conformance tests



\## Proposal Process



Major protocol changes must be submitted as a proposal in:



proposals/



Example:



DIP-0001-core-artifact-format.md
